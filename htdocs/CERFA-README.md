# Formulaires CERFA à ajouter

Pour que les formulaires CERFA fonctionnent correctement, vous devez ajouter les images suivantes dans le dossier `JDC/public/` :

## Fichiers requis :

1. **cerfa-13750.jpg** - Image du formulaire CERFA 13750 (Certificat d'immatriculation)
2. **cerfa-13757.jpg** - Image du formulaire CERFA 13757 (Mandat à un professionnel)

## Optionnel (pour le téléchargement PDF) :

3. **cerfa-13750.pdf** - Version PDF du formulaire CERFA 13750
4. **cerfa-13757.pdf** - Version PDF du formulaire CERFA 13757

## Où trouver les formulaires :

- Site officiel de l'ANTS : https://www.ants.gouv.fr/
- Service-public.fr : https://www.service-public.fr/particuliers/vosdroits/F31228

## Instructions :

1. Téléchargez les formulaires CERFA depuis le site officiel
2. Convertissez-les en images JPG (ou utilisez des captures d'écran)
3. Placez les fichiers dans le dossier `JDC/public/`
4. Les formulaires apparaîtront automatiquement sur la page Administrative
